## Resumen del Proyecto
Este repositorio contiene la implementación de automatización y auditoría para la empresa Farmacia Central SpA. El proyecto se dividió en 5 fases principales:

1. **Fase 1 (Baseline):** Captura del estado inicial del router usando PyATS/Genie.
2. **Fase 2 (Aprovisionamiento):** Configuración corporativa mediante Ansible garantizando idempotencia.
3. **Fase 3 (Validación NETCONF):** Auditoría de configuraciones extrayendo el modelo YANG en XML.
4. **Fase 4 (Validación RESTCONF):** Auditoría de endpoints específicos mediante peticiones HTTP/JSON.
5. **Fase 5 (Compliance):** Generación de reporte final y Diff comprobando las modificaciones aplicadas.

**Estado Final de la Implementación:** CONFORME.
